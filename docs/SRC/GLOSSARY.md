# Glosario de Términos

**Proyecto:** synapse-agentic  
**Fecha:** 2026-03-21

---

## Términos

| Término | Definición |
|---------|------------|
| Agentic System | Sistema AI que puede tomar acciones autónomas |
| MCP | Model Context Protocol - Protocolo para integrar AI assistants |
| Actor Model | Modelo de concurrencia con mensajes entre actores |
| Consensus Voting | Mecanismo para tomar decisiones entre múltiples LLMs |
| Tool | Capacidad definida como JSON-Schema que el agente puede ejecutar |
| Memory Store | Almacenamiento pluggable para memoria de corto/largo plazo |
| Tokio | Runtime async para Rust |
| Supervisor | Componente que supervisa y reinicia agentes fallidos |
